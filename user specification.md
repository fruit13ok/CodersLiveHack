# Coders Live Hack Project

## User specification

### Target users
* rookie programmers

### User gets the following from this app

Coders Live Hack is a self improve productivity tracker app for inspired by the Pomodoro Technique. It helps rookie  programmers measure productivity, and track tasks. The application  measures coders’ learning by keeping track of their time spent when working on their programming projects, similar to Pomodoro Technique. It retrieves commit data from their GitHub account to track their productivity. Coders’ productivity is displayed as a graph so they can see their progress easily.
CodersLiveHack’s goal is to inspire new coders while finding out their strengths and weaknesses as a programmer.
